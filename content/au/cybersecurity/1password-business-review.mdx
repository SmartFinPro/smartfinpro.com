## Key Features for Australian Finance Teams

### 1. Zero-Knowledge Security Architecture

1Password uses a dual-key encryption model that ensures maximum security:

| Feature | Specification |
|---------|--------------|
| Encryption | AES-256-GCM |
| Key Derivation | PBKDF2 with 650,000 iterations |
| Secret Key | 128-bit locally generated |
| Transport Security | TLS 1.3 with certificate pinning |

<Info>
**Secret Key + Master Password:** 1Password's dual-key model means that even if your master password is compromised, attackers still need your Secret Key to decrypt vault data. This provides significantly stronger protection than single-factor password managers.
</Info>

### 2. Enterprise Admin Console

The admin dashboard provides Australian IT teams with comprehensive control:

- **User provisioning:** SCIM-based automated onboarding and offboarding
- **Custom groups:** Organise teams by department, project, or compliance boundary
- **Vault policies:** Enforce password strength, MFA, and sharing rules
- **Activity logs:** APRA-ready audit trails with exportable reports
- **Usage reports:** Monitor adoption rates and security posture
