## What is Perimeter 81?

Perimeter 81 is a cloud-native Zero Trust Network Access (ZTNA) platform that replaces traditional VPN appliances with modern, software-defined security. Unlike legacy VPNs that grant broad network access, Perimeter 81 enforces least-privilege access at the application level, ensuring users only reach the specific resources they need.

For Canadian financial services, Perimeter 81 delivers:
- **Zero Trust Network Access** with per-application security policies
- **Cloud-native deployment** with no hardware requirements
- **PIPEDA and OSFI alignment** with comprehensive audit logging
- **Automatic Wi-Fi security** for remote and travelling teams
- **Network segmentation** enforcing least-privilege access principles

## Key Features for Canadian Finance Teams

### 1. Zero Trust Network Access (ZTNA)

Perimeter 81's core architecture enforces security at every access point:

| Feature | Specification |
|---------|--------------|
| Access model | Per-application, least-privilege |
| Authentication | Identity-based with MFA enforcement |
| Device posture | Health checks before granting access |
| Encryption | AES-256 with WireGuard protocol |
| Micro-segmentation | Application-level network isolation |

<Info>
**Zero Trust vs Traditional VPN:** A traditional VPN gives users broad network access once authenticated. Perimeter 81's ZTNA verifies every access request against identity, device health, location, and time-of-day policies. If a device is compromised, the blast radius is limited to the specific application being accessed.
</Info>

### 2. Software-Defined Perimeter (SDP)

Perimeter 81 creates invisible network boundaries around your applications:

- **Network cloaking:** Resources are invisible to unauthenticated users
- **Dynamic access:** Policies adjust based on user context and risk signals
- **Split tunnelling:** Route only business traffic through secure channels
- **Multi-cloud support:** Secure access to AWS, Azure, GCP, and on-premises
- **Agentless access:** Browser-based access for specific applications
